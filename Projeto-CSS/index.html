Estilo CSS(Cascading Style Sheets)

Existem 3 usos pro CSS:

  • External Style Sheet
  • Internal Style Sheet
  • Inline Style

O estilo externo consiste em criar um arquivo main.css para colocar as alterações de estilo nos elementos que desejamos. Esse é o recomendado por critérios de organização, para simplicidade do código e reuso.

O estilo interno consiste em alterações diretamente no head e aplicar.

Basicamente, nessa aula é ensinado a utilizar o CSS Inline, isto é, podemos alterar o estilo (realizar as formatações) direta no elemento que quero que seja alterado.

Para usar o css nós definimos uma propriedade e seu valor, assim isso constitui uma declaração!

  • Por exemplo : style="color: purple;"
  • Por exemplo : style="font-style: oblique;"
  • Por exemplo : style="font-family: fantasy;"

Aula 02 - Internal Style Sheet

Uso de seletores, dentro do Head nós podemos adicionar folhas de estilo, javascript, etc.

Com o uso de seletores, para casos onde existem muitos parágrafos, n seria viável adicionar a personalização inline, visto que demandaria mais esforço para a busca de um mesmo resultado. Assim, com o internal style sheet, usamos os seletores, classes e ID's para fazer essa diferenciação.

Seletores será o nome da tag que queremos formatar, dentro da classe colocaremos a declaração com uma propriedade que recebe um valor

Por exemplo, as cores dos parágrafos estão no expectro RGB (4,4,51) alternadas na head do arquivo, bem como as tags h2 e h1.
